Ordering VSSEx

If you are using VSSEx for personal use, you are not required to purchase a license.  For businesses using VSSEx, a license must be purchased for each workstation on which VSSEx is installed.

Be sure you read the EULA before purchasing VSSEx.

Pricing

A single VSSEx license can be purchased for $5 US.  The license purchased allows business use of VSSEx per-workstation.  Large-quantity pricing arrangements can be made.
